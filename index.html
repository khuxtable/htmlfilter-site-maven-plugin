Introduction

The HTML Filter Site Maven plugin filters HTML files destined for site documentation in two ways:

  • Tidy - It can run JTidy on them to clean them up and make them XHTML compatible.

  • Merge - It can run them through a Velocity template to provide the appearance of a Doxia-generated file without Doxia's habit of stripping useful formatting.

Goals Overview

The HTML Filter Site plugin has two goals:

  • htmlfilter-site:merge runs Velocity on all files in a directory matching a pattern.

  • htmlfilter-site:tidy runs JTidy on all files in a directory matching a pattern.
